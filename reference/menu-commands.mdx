LongFred organises its commands into two layers: the **Main Menu**, which you open from any throttle screen by pressing the **Menu** button, and the **Extras sub-menu**, which is item 9 inside the Main Menu. In addition, a **Direct Commands** screen is available from the throttle screen itself — press **OK** while a loco is active to open it without going through the menu. This page lists every option in all three layers.

## How to Navigate the Menus

<Steps>
  <Step title="Open the Main Menu">
    Press the **Menu** button (GPIO 10) from the throttle screen. The display switches to the menu list.
  </Step>
  <Step title="Select a command">
    Use the **Up / Down** buttons to move the highlight to the command you want, or press the **function key** (F0–F9, corresponding to digits 0–9) that matches the item number shown on screen to jump directly to that item. Then press **OK** to execute the command or open any required sub-screen.
  </Step>
  <Step title="Return or cancel">
    Press **Back** at any time to cancel the current action and return to the previous screen. From the Main Menu, Back returns you to the throttle screen without executing any command.
  </Step>
</Steps>

<Note>
Menu items are zero-indexed: the first item in each menu is **0**, not 1. The **0** key (F0 button) or the Up/Down cursor can select the first item.
</Note>

## Main Menu (Menu button → select 0–9 → OK)

| Item | Command | What it does |
|---|---|---|
| **0** | Function | Toggle a DCC function on the active loco; prompts for a function number |
| **1** | Add Loco | Acquire a new locomotive address onto the current throttle slot |
| **2** | Drop Loco | Release the active locomotive from the current throttle slot |
| **3** | Toggle Dir | Reverse the direction of the active locomotive |
| **4** | X SpeedStep | Cycle the speed-step multiplier through 1×, 2×, and 4× |
| **5** | Throw Point | Send a Throw (diverge) command to a turnout/point; prompts for address |
| **6** | Close Point | Send a Close (straight) command to a turnout/point; prompts for address |
| **7** | Route | Activate a predefined route on the command station; prompts for route ID |
| **8** | Trk Power | Toggle track power on or off at the command station |
| **9** | Extras | Open the Extras sub-menu |

<Tip>
**Trk Power** (item 8) is the quickest way to restore motion when all locos are stationary unexpectedly — it toggles the track power state without needing to know loco addresses.
</Tip>

## Extras Sub-Menu (Main Menu → 9 → Up/Down → OK)

The Extras sub-menu contains device configuration and session management options. Use **Up / Down** to highlight an item, then press **OK** to activate it.

| Display label | Command | What it does |
|---|---|---|
| **0 Fnc/Key Tgl** | Fnc/Key Tgl | Toggle between showing the Direct Commands screen or the Function List when you press OK on the throttle screen |
| **1 Net Config** | Net Config | Open the network configuration screen to set SSID, password, server IP, and port |
| **2 Device** | Device | Show device information (firmware version, MAC address, etc.) |
| **3 Heartbt Tgl** | Heartbt Tgl | Enable or disable the periodic WiThrottle heartbeat (default period: 10 s) |
| **4 #Throttles +** | #Throttles + | Increase the number of active throttle slots (up to a maximum of 6) |
| **5 #Throttles –** | #Throttles – | Decrease the number of active throttle slots (minimum 1) |
| **6 Language** | Language | Cycle through available UI languages |
| **7 OFF / Sleep** | OFF / Sleep | Put the device to sleep; press the encoder button (GPIO 0) to wake |
| **8 1 Loco Tgl** | 1 Loco Tgl | Toggle single-loco mode, which constrains operation to one locomotive at a time |
| **9 Save Locos** | Save Locos | Persist the current locomotive roster (addresses and names) to flash storage |

<Warning>
Pressing **OFF / Sleep** (Extras → 7) immediately suspends the display and radio. Any running locomotive will continue at its last commanded speed — set all locos to zero before sleeping if the layout is unattended.
</Warning>

<Info>
The default number of throttle slots is **2**. Use **#Throttles +** and **#Throttles –** to adjust between 1 and 6 slots. Changes take effect immediately but are lost on reboot unless you also run **Save Locos**.
</Info>

## Direct Commands (Throttle Screen → OK)

When a loco is active on the throttle screen, pressing **OK** opens the Direct Commands screen. Use **Up / Down** to highlight the action you want, then press **OK** again to execute it. This screen is designed for the most frequently needed actions during a running session.

| Display label | Action | What it does |
|---|---|---|
| **0-4 Function** | Function | Toggle DCC function F0 on the active loco (typically headlights) |
| **5 NextThr** | Next Throttle | Advance focus to the next throttle slot |
| **6 SpdMult** | Speed Multiplier | Cycle the encoder speed-step multiplier (1× → 2× → 4× → 1×) |
| **7 Rev** | Reverse | Set the active loco's direction to reverse |
| **8 EStop** | EStop | Issue an emergency stop to the active loco (speed set to 0 immediately) |
| **\* back** | Back | Return to the throttle screen without executing a command |

<Note>
The numbers shown in the **Display label** column are informational labels that appear on screen — they reflect the key assignments shown in the interface. Navigation on the Direct Commands screen is always by Up/Down and OK.
</Note>

<Tip>
If you want Function List access instead of Direct Commands when pressing OK on the throttle screen, toggle the mode via **Extras → 0 (Fnc/Key Tgl)**.
</Tip>
