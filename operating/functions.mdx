LongFred provides eleven physical function buttons, labelled **F0** through **F10**, each directly mapped to the corresponding DCC function number on your locomotive decoder. Pressing a button sends a "function on" command; releasing it sends "function off". This momentary behaviour matches the default DCC convention for non-latching functions. Function buttons only send commands when at least one loco is acquired on the current throttle slot.

## Function button reference

| Button | DCC function | Typical use |
|--------|-------------|-------------|
| **F0** | Function 0 | Headlight (applied to **all** locos in a consist) |
| **F1** | Function 1 | Bell (lead loco only in a consist) |
| **F2** | Function 2 | Horn / whistle (lead loco only) |
| **F3–F10** | Functions 3–10 | Aux lights, sound effects, couplers, etc. (lead loco only) |

<Note>
  The actual function labels — such as "Bell", "Whistle", or "Cab Light" — depend on how the loco is configured in the server roster, not on the button label itself. LongFred downloads the labels from the server when you acquire a roster entry.
</Note>

## Viewing function labels and states

You can see the full list of function labels and which ones are currently active on the **Function List** screen:

<Steps>
  <Step title="Open the menu">
    Press **Menu** from the throttle screen.
  </Step>
  <Step title="Select Function">
    Press **0** (or navigate to **0 Function** and press **OK**). The Function List screen opens, showing ten functions per page. Each entry displays the function number, its label (loaded from the server), and an indicator showing whether it is currently on.
  </Step>
  <Step title="Navigate pages">
    Use the page-forward key to move through pages and see functions **F0–F31**. LongFred supports up to 32 functions in the list, though only F0–F10 have dedicated physical buttons.
  </Step>
</Steps>

## Function behaviour in a consist

When you have multiple locos on a single throttle slot (a consist), function routing works as follows:

<CardGroup cols={2}>
  <Card title="F0 — broadcast to all locos" icon="broadcast-tower">
    F0 (typically the headlight) is sent to **every loco** in the consist simultaneously, so all headlights turn on and off together.
  </Card>
  <Card title="F1–F10 — lead loco only" icon="train-front">
    All other function buttons target the **lead loco** only. Sounds and non-headlight lights on the trailing locos are controlled by their individual decoders and are not affected by these button presses.
  </Card>
</CardGroup>

## Fnc/Key toggle — function list vs direct commands

The **OK** key on the throttle screen (when a loco is acquired) can open either the **Function List** or the **Direct Commands** screen. By default it opens the Direct Commands screen.

To switch between the two behaviours:

1. Press **Menu → 9 Extras**
2. Navigate to **0 Fnc/Key Tgl** and press **OK** to toggle the mode.

When the toggle is set to show functions, pressing **OK** from the throttle screen goes straight to the Function List.

## Direct Commands screen

When the mode is set to direct commands (the default), pressing **OK** on the throttle screen opens the Direct Commands screen. Use the navigation joystick or encoder to highlight an item, then press **OK** to execute it:

| Display label | Action |
|---------------|--------|
| `0-4 Function` | Trigger DCC Function 0 on the current loco |
| `5 NextThr` | Switch to the next active throttle slot |
| `6 SpdMult` | Cycle the speed step multiplier (1×, 2×, 4×) |
| `7 Rev` | Set direction to Reverse on the current loco |
| `8 EStop` | Emergency stop all locos on all throttles |
| `* back` | Return to the throttle screen |

<Tip>
  The Direct Commands screen gives you quick access to EStop and Next Throttle without navigating the menu. The dedicated hardware EStop button always works regardless of which screen you are on.
</Tip>
