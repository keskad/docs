The throttle screen is the main operating view you spend most of your time on. It packs everything you need to drive a locomotive into a single compact OLED display, organised into distinct regions so you can read key information at a glance without taking your eyes off the layout for long.

## Screen layout overview

<CardGroup cols={2}>
  <Card title="Throttle slot number" icon="hashtag">
    Displayed in the top-left corner. Shows which of the active throttle slots you are currently controlling — for example `0` for the first slot, `1` for the second, and so on. LongFred supports up to six slots; two are active by default.
  </Card>
  <Card title="Speed" icon="gauge-high">
    The current DCC speed step for the active slot, shown as a number from **0** to **126** using the DCC 128-step scale. A value of `0` means the loco is stopped.
  </Card>
  <Card title="Direction indicator" icon="arrows-left-right">
    Shows **F** (Forward) or **R** (Reverse) — the direction currently sent to the lead loco on this throttle slot.
  </Card>
  <Card title="Loco address area" icon="train">
    Lists the DCC address (or addresses) of every locomotive currently acquired on this throttle slot. Short addresses are shown as `S`*n* (e.g. `S42`) and long addresses as `L`*n* (e.g. `L3452`). Multiple locos in a consist appear side by side separated by spaces.
  </Card>
  <Card title="Track power icon" icon="bolt">
    Indicates whether track power is currently on or off as reported by the command station. Use **Menu → 8 Trk Power** to toggle it.
  </Card>
  <Card title="Heartbeat indicator" icon="heart-pulse">
    Shows the state of the WiThrottle heartbeat. LongFred sends a keep-alive to the server every 10 seconds by default. You can toggle heartbeat transmission from **Menu → Extras**, then navigate to **3 Heartbt Tgl** and press **OK**.
  </Card>
  <Card title="Function state row" icon="toggle-on">
    A compact row showing the on/off state of DCC functions **F0 through F10** for the active throttle slot. An illuminated indicator means that function is currently on. Press any of the eleven function buttons to toggle them.
  </Card>
  <Card title="Battery icon" icon="battery-three-quarters">
    Appears in the top-right corner when battery monitoring is enabled. Cycles between icon-only and icon-with-percentage modes via the battery display option in Extras. Hidden when battery monitoring is disabled.
  </Card>
</CardGroup>

## The 'No loco selected' state

When a throttle slot has no locomotive acquired, the address area displays **No loco selected** instead of a DCC address. Speed and direction commands have no effect in this state; the encoder and function buttons are inactive for that slot.

To start driving, you need to acquire a loco. With the throttle screen showing *No loco selected*, you can type a DCC address directly using the **F0–F9** keys as a numeric keypad and press **OK** (or the encoder button) to acquire it. Alternatively, press **Menu → 1 Add Loco** to pick a locomotive from the server roster.

<Note>
  Switching to a different throttle slot with **Next Throttle** (navigate to **5 NextThr** on the Direct Commands screen and press **OK**) lets you control a second or third locomotive independently without losing the one you already acquired.
</Note>

## The footer hint bar

The bottom line of the throttle screen shows a context-sensitive hint:

```
Menu OK Fn  EStop
```

This tells you the four primary shortcuts available while you are on the throttle screen:

| Label  | Meaning                                                                 |
|--------|-------------------------------------------------------------------------|
| `Menu` | Open the main menu to add/drop locos, set track power, and more        |
| `OK`   | Confirm a loco address you have started typing, or open/confirm an item |
| `Fn`   | Press a function button (F0–F10) to toggle the corresponding DCC function |
| `EStop`| Send an emergency stop to all locos on all active throttle slots immediately |

## Broadcast messages

If the connected command station or server sends a broadcast text message, it temporarily replaces the footer hint on the throttle screen. The message is displayed for **10 seconds**, after which the normal `Menu OK Fn  EStop` hint reappears automatically. No action is required to dismiss it.

<Info>
  Broadcast messages also appear as the footer on other screens (roster list, menu, etc.) while they are active, so you will not miss them even if you have navigated away from the throttle screen.
</Info>
