LongFred is a physical wireless throttle controller for model railway layouts. It connects to your DCC command station over WiFi using the WiThrottle or Z21 protocol, letting you drive locos, control turnouts and routes, and manage track power — all from a handheld device with an OLED display, rotary encoder, and function buttons.

## What LongFred does

LongFred turns an ESP32-C6 development board into a full-featured wireless throttle. It speaks the WiThrottle protocol (compatible with JMRI, DCC-EX, and other servers) and the Roco Z21 protocol, so it works with the most popular DCC command stations available today.

## Get started in three steps

<Steps>
  <Step title="Connect to your layout WiFi">
    Power on LongFred, select your network from the SSID list on the display, and enter the password using the function keys and navigation buttons.
  </Step>
  <Step title="Find your command station">
    LongFred uses mDNS to discover WiThrottle servers automatically. Select your server from the list, or enter its IP address and port manually for Z21.
  </Step>
  <Step title="Acquire a loco and drive">
    Enter a DCC address on the throttle screen or pick a loco from your server roster. Turn the encoder to set speed, press the Direction button to reverse.
  </Step>
</Steps>

<Tip>
  If your command station is a DCC-EX system running as a WiThrottle server, LongFred will find it automatically the moment it appears on the network — no configuration needed.
</Tip>
